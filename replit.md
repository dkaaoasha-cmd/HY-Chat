# 한양챗 (HY-Chat) - 한양대학교 AI 비서 PoC

## 프로젝트 개요
한양대학교 학생들을 위한 AI 맞춤형 비서 서비스 PoC입니다. Streamlit과 Python을 사용하여 구축되었습니다.

## 핵심 기능

### 1. AI 학사 챗봇
- 학사 관련 질문에 대한 답변 제공
- 5개 카테고리: 수강신청, 기숙사, 학사 행정, 졸업, 등록금·장학
- 출처 링크 및 관련 규정 카드 표시

### 2. 맞춤형 장학금 추천
- 사용자 프로필 기반 장학금 매칭
- 매칭 점수를 진행 바로 시각화
- 상세보기 기능으로 선발 요강 및 신청 기한 확인
- **장학금 신청 일정 캘린더**: 
  - 구글 캘린더 스타일 월별 그리드 뷰
  - 장학금 마감일을 날짜별로 표시
  - 유형별 색상 구분 (성적, 소득연계, 근로, 외부, 특별)

### 3. 전공 진로 로드맵
- 희망 진로별 필수 역량 vs 보유 역량 레이더 차트
- 한양대학교 실제 개설 과목 추천 (과목코드, 학점, 개설학과, 과목설명 포함)
- 부족 역량 보완을 위한 추천 대외활동
- 타임라인 형태의 성장 로드맵
- 총 학점 계산 표시
- **공모전·인턴·대외활동 캘린더**: 
  - 구글 캘린더 스타일 월별 그리드 뷰
  - 월 네비게이션 (이전 달/다음 달, 오늘로 이동)
  - 요일 헤더 (월~일) 및 날짜별 활동 배지 표시
  - **여러 날에 걸친 이벤트는 연결된 막대로 표시** (시작-중간-끝 연결)
  - 오늘 날짜 강조 표시
  - 활동 유형별 필터 (공모전, 인턴십, 대외활동, 서포터즈, 해커톤)
  - 마감일/시작일/이름순 정렬
  - 외부 플랫폼 링크 (링커리어, 위비티, 캠퍼스픽, 슥삭, 자소설닷컴)
  - 상세 정보 (주최, 기간, 대상, 혜택, 분야, 링크)

## 기술 스택
- **Framework**: Streamlit
- **시각화**: Plotly (레이더 차트, 타임라인)
- **데이터**: 더미 데이터 시뮬레이션

## 실행 방법
```bash
streamlit run app.py --server.port 5000
```

## 디자인 가이드
- 메인 테마 색상: #0E4A84 (한양대학교 Blue)
- 탭 기반 네비게이션으로 세 가지 기능 전환
- 스피너로 AI 구동 느낌 제공

## 파일 구조
